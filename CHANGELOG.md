# Changelog

## 1.1.0

- Added admin-level "Default Segment for All Users" setting in General Settings
- Users without a personal default segment now fall back to the admin-configured default
- Personal default segments continue to take priority over the admin default
- Updated inline help to explain the fallback behavior
- Added English and French translations for new settings

## 1.0.0

- Initial release
- Per-user default segment setting in Personal Settings
- Automatic segment application when viewing reports
- Does not override manually selected segments
- Graceful transition when changing the default segment
- English and French translations
- Inline help with use case example in the admin UI
